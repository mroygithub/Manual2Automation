What is Web Services[API] & its types?

A Web service is a group of organized code that can be remotely invoked using HTTP. Web Services allows different applications to talk to each other and share data and services among themselves. As an example , placing an Amazon order through Android / iphone / Desktop browsers - All are different platforms but middle layer Web Service are same.
An API is a set of definitions and protocols for building and integrating application software. API stands for application programming interface.API testing allows the user to test headless technologies like JMS, HTTP Databases, and Web Services.
There are two types of web service:
SOAP or Simple Objects Access Protocol-is a web communication protocol used to expose web services and transmit data over HTTP/HTTPS.Supports the XML data format only.
REST or RESTful API design (Representational State Transfer)- when a RESTful API is called, the server will transfer to the client a representation of the state of the requested resource. The representation of the state can be in a JSON format, and probably for most APIs this is indeed the case. It can also be in XML or HTML format.

Rest Assured GET Request


1. Once maven project is created and Rest Assured dependency is added. Right click on src/test/java --> Create a package-->Create a simple java class with main() function.




2. Open Chrome --> Right Click and Inspect window --> CLick network tab --> Now hit https://www.google.com , you will find below screen in network tab. We are going to validate same using Rest Assured.


3. Check below screenshot where
Request URL: https://www.google.com/
Request Method: GET
Status Code: 200 





    
    package RestAssured;
    
    import io.restassured.RestAssured.*;
    import io.restassured.matcher.RestAssuredMatchers.*;
    import org.hamcrest.Matchers.*;
    import io.restassured.RestAssured;
    import io.restassured.response.Response;
    
    public class RestTest {


        public static void main(String[] args) {

            // Validate GET request...
            RestAssured.baseURI="http://www.google.com"; // To initiate the service...
    
            Response res = RestAssured.get(); // Dot Get means we are hitting a Get request same as Chrome..
            // How to validate Status code and its response ???
    
            System.out.println("Status Code is =="+ res.getStatusCode());
        }
        
    }


4. After we run the code in Eclipse , Status Code: 200 as output




5. In Chrome network tab click on Response and see response body is there. Check below screen.




6. In Eclipse just ADD res.getBody().asString() , it would print the response.

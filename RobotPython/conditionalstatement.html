How To Use Conditional Statement In Robot Framework

if ...... Else Statement in Robot Framework

Test Where If statement is getting executed

Consider The Below Code ...

*** Settings ***
   Documentation  Wikipedia Conditional Testing [IF]
   Library  SeleniumLibrary
   
   *** Variables ***
   
   *** Test Cases ***
   Execute IF Statement
       Open Browser  https://www.wikipedia.org/  Chrome
       ${countElement}=  Get Element Count  xpath://span[text()='Wikivoyage']
       Run Keyword If  ${countElement} > 0  Click Wikivoyage Link  # If the element is present we will get the value of count as 1
       ...  ELSE  Click Google Link
       Title Should Be  Wikivoyage  timeout=3
       Close Browser   
   
   *** Keywords ***
   Click Wikivoyage Link
       Click Element  xpath://span[text()='Wikivoyage']
   
   Click Google Link
       Click Element  xpath://span[text()='Google']

In above example it is given --
${countElement} Variable to get count of Xpath
Get Element Count Keyword to get Xpath Count
Run Keyword If Keyword to run if statement
Run Keyword If ${countElement} > 0 Click Wikivoyage Link Calling Click Wikivoyage in case if condition is satisfied.
... ELSE Click Google Link In case condition not satisfied then call

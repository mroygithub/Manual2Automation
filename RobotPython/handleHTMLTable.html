Robot Framework : Handle HTML Table

HTML Table row/column/cell/header.... in Robot Framework

Robot : How To Read HTML Table ?

Consider The Below HTML Table

How to get all row count?

${rows}= get element count xpath://table[@name='BookTable']/tbody/tr

How to get all column count?

${cols}= get element count xpath://table[@name='BookTable']/tbody/tr[1]/th

Read HTML Data By Xpath: Second Row and First Column

${dataRead}= get text xpath://table[@name='BookTable']/tbody/tr[2]/td[1]

4 th Column should have value as 3000

table column should contain xpath://table[@name='BookTable'] 4 3000

5 th row should have value as 'Master In Selenium'

table row should contain xpath://table[@name='BookTable'] 5 Master In Selenium

3 rd row and 2nd column should have value as 'Mukesh'

table cell should contain xpath://table[@name='BookTable'] 3 2 Mukesh

Table Header should have column name as 'Subject'

table header should contain xpath://table[@name='BookTable'] Subject

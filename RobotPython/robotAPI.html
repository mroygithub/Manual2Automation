Robot : Perform POST API CALL

Step 1: Using postman do a post API call as below

From POSTMAN screenshot we can say:
Request URL: http://dummy.restapiexample.com/api/v1/create
Request Body: {"name":"name_001","salary":"53001","age":"23"}
Response Body status: { "status": "success", "data": { "name": "name_001", "salary": "53001", "age": "23", "id": 5379 }, "message": "Successfully! Record has been added." }

Same POSTMAN POST operation can be perform through Robot Framework as below-

*** Settings ***
   
   Library     REST
   
   *** Variables ***
   
   ${testBody}     {"name":"name_001","salary":"53001","age":"23"}
   
   *** Test Cases ***
   
   
   Do a GET Request For jsonplaceholder
   
       #POST   http://dummy.restapiexample.com/api/v1/create    ${CURDIR}/apijson.json
       POST   http://dummy.restapiexample.com/api/v1/create    ${testBody}
       Integer     response status           200
       Output    response      body
       String response body status    success
       String response body message    Successfully! Record has been added.

Explaining above code below --
PO${testBody}ST Variable to store request JSON body
{"name":"name_001","salary":"53001","age":"23"} request JSON body
POST Keyword To Do A POST CALL
http://dummy.restapiexample.com/api/v1/create Request URL
#POST http://dummy.restapiexample.com/api/v1/create ${CURDIR}/apijson.json This is commented line , showing to pass .JSON file from project Directory.
POST http://dummy.restapiexample.com/api/v1/create ${testBody} POST call with request BODY
response response for the GET call
status This is to check status code.
200 Expected response status code
String response body status success To validate status='success' and it is String

from xml.dom import minidom
      from pathlib import Path
      
      
      def read_xml(node_name):
      
            # parse an xml file by name
          my_doc = minidom.parse(str(Path(__file__).parent.parent) + '/XML_Data.xml')
            data = my_doc.getElementsByTagName(node_name)[0]
            return data.firstChild.data

In above example it is given --
from xml.dom import minidom Package to work on XML File.
from pathlib import Path Package to get current project directory path.
minidom.parse(str(Path(__file__).parent.parent) + '/XML_Data.xml') XML path using current project directory.
return data.firstChild.data This will return XML node value.

Step4 : Write below python code to read JSON file 

      import json
      from pathlib import Path
      
      
      def readjsonaspertest(testcase , param):
            with open(str(Path(__file__).parent.parent)+'/JSON_Data.json') as f:
               json_data = json.load(f)
      
               for data in json_data['HomePage']:
                  element = (str(data[testcase])).split(",")
                  for x in element:
                        #print(x)
                      if(param in x):
                           break
                  return x.split(":")[1].replace('"','')

In above example it is given --
import json Package to work on JSON File.
from pathlib import Path Package to get current project directory path.
with open(str(Path(__file__).parent.parent)+'/JSON_Data.json') as f: To capture JSON data into 'f'.
json_data = json.load(f) To load JSON data 'f' into variable 'json_data'
for data in json_data['HomePage']: To run a for loop to get data under parent node 'HomePage'
element = (str(data[testcase])).split(",") Splitting data by ',' as per 'testcase' and store it into 'element'
for x in element: Run a for loop for list of 'element'
if(param in x): Searching for 'param' in x.
return x.split(":")[1].replace('"','') This will return value of 'param'

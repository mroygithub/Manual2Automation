How Can we create user define keyword in Robot Framework

HOW TO WRITE USER DEFINED KEYWORDS IN ROBOT FRAMEWORK?

Consider below code where we are doing below activities -


1. Open Browser https://www.google.com Chrome – Launch Google in Chrome Browser
2. Maximize Browser Window – Maximize the Chrome Browser
3. Wait For Condition return document.title == "Google" – Validate Google Title
4. Wait Until Page Contains Element xpath://img[@class='lnXdpd'] – Validate Google Logo
5. Element Should Be Visible xpath:(//input[@name='btnK'])[2] – Validate Google Search button
6. Close All Browsers – Close Application

*** Keywords ***

Now we can break the above steps with custom keys under *** Keywords *** section as below

*** Keywords ***

Validate Google Page Title
    Wait For Condition  return document.title == "Google"

Validate Google Logo
    Wait Until Page Contains Element    xpath://img[@class='lnXdpd']

Validate Google Search Button
    Element Should Be Visible    xpath:(//input[@name='btnK'])[2]

Close My Application
    Close All Browsers

So Keys are described as below -


1. Validate Google Page Title – Validate Google Title
2. Validate Google Logo – Validate Google Logo
3. Validate Google Search Button – Validate Google Search button
4. Close My Application – Close Application

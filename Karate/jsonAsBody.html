Karate Read JSON : Example :1 [Feature File]


In below code 'readJsonAsBody' is a JSON object 


        #########   https://manual2Automation.com ###############
          #######       Created By :: MITHUN ROY ################
        
        Feature: User wants to check Google URL as GET
        
        
            Background:
        
            * def serviceURL = 'http://dummy.restapiexample.com/api/v1/create'
        
        
         # Scenario 2 :: Example of GET
        
          Scenario Outline: Verify that a new employee is successfully getting created
        
            * def readJsonAsBody = {"name":'<Name>',"salary":<Salary>,"age":<Age>}
        
            Given url serviceURL
            When request readJsonAsBody
            And method post
            Then status 200
            And match responseType == 'json'
            And print 'Response is: ', response
            And response.status == '<Status>'
            And response.data.name == '<Name>'
            And response.data.salary == <Salary>
            And response.data.age == <Age>
        
            Examples:
            |Name|Salary|Age|Status|
            |MIRHUN|5000|30 |success|
            |ARUN  |10000|25|success|
            |UMA   |2000 |12|success|

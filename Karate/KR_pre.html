Karate Prerequisite

Karate, a Behavior Driven Development (BDD) testing framework for Java , is built on top of Cucumber. However, unlike Cucumber, tests aren't written in Java and are fully explained in the Gherkin file.
Below are prerequisite for Karate:


1. Java / JADK - Latest Version
2. IntelliJ / Eclipse
3. karate-apache Dependency [https://mvnrepository.com/artifact/com.intuit.karate/karate-apache]
4. karate-junit4 Dependency [https://mvnrepository.com/artifact/com.intuit.karate/karate-junit4]

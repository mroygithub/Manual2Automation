        Feature: User wants to check Google URL as GET
        
        
            Background:
        
            * def urlGoogle = 'https://www.google.com'
        
        
         # Scenario 1 :: using urlGoogle from Background
        
          Scenario: Testing valid GET endpoint
        
            Given url urlGoogle
            When method GET
            Then status 200
            And match responseType == 'string'
            And assert responseTime < 1000


This this code we are validating --
Response code is 200
Response type is string type of data.
Response time is less than 1000.

What is Karate Background?




        #########   https://manual2Automation.com ###############
          #######       Created By :: MITHUN ROY ################
        
        Feature: User wants to check Google URL as GET
        
        
            Background:
        
            * def expectedResult = 'Welcome 10'
            * def urlGoogle = 'https://www.google.com'
        
          # Scenario 1 :: Validating expectedResult from Background
        
          Scenario: Testing String Concat one
        
            Given def YouAre = 'Welcome '
            And def num = 10
            Then assert YouAre + num == expectedResult
        
         # Scenario 1 :: using urlGoogle from Background
        
          Scenario: Testing valid GET endpoint
        
            Given url urlGoogle
            When method GET
            Then status 200

From above Feature File 2 scenarios we can see ,
expectedResult is declared in Background and used from the scenario.
urlGoogle is declared in Background and used from the scenario.

          Scenario: Verify that a new employee is successfully getting created
        
            Given url 'http://dummy.restapiexample.com/api/v1/create'
              When request requestBody
              And method post
              Then status 200
              And print 'Response is: ', response

In above code we can see we are calling a POST request with an XML file as request body. This request body is declared in Background

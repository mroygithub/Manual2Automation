Below screenshot shows Karate basic Framework Architecture. It has :
src/main/java == Not useful initially.
src/test/java == It is important. We have created 3 packages under it.
FeatureFiles -> For keeping all feature files.
testRunner.com -> For keeping test runner file.
supportingjson -> For keeping any supporting files.
target -> Default KARATE reporting.
pom.xml -> This is for dependency and other plugins.

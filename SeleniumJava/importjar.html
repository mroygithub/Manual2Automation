Eclipse : Import Selenium Jar

Import Selenium jar


1. If Selenium Jar is not downloaded then download it from SeleniumHQ.org




2. Now Jar is available at Download folder




3. Once project is created in Eclipse , right click on project -->Build Path--> Configure Build Path , see screenshot below




4. Click on Library TAB then click on 'Add external Jars'




5. Select Selenium Webdriver Jar where it is already downloaded.




6. Once it is downloaded then we can see Selenium Jar in project path. Check below screenshot.

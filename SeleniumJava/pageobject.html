Page Object Model

Page Object Model is type of implementation where we can store Web page locators in a structural way.



Page Object Model : Using "String" return type

Below picture presents a page object model with below informations.

package pageobjectmodel;
Under this package we have 3 class files represents 3 different pages.
homepage
loginpage
registrationpage
homepage details with 3 web locators are given below. 



            
            package pageobjectmodel;
            
            import org.openqa.selenium.By;
            
            
            public class homepage {
            
            
                public String google_logo()
                {
                    return "hplogo";
                }
            
                public String google_searchTextBox()
                {
                    return "q";
                }
            
                public String google_search_button()
                {
                    return "btnK";
                }
            
            
            }


3 Web loactors are represents with 3 methods having String return type.
google_logo
google_searchTextBox
google_search_button


We can use these locators as below, where we are creating object of the page class and calling the methods 

    package SelJava;
    
    import org.openqa.selenium.By;
    import org.openqa.selenium.WebDriver;
    import org.openqa.selenium.chrome.ChromeDriver;
    import pageobjectmodel.homepage;
    
    public class testScripts1 {
    
    
    
        public static WebDriver driver = null;
        private static homepage HPobj;
    
        public static void main(String args[]) throws Exception
    
        {
            HPobj = new homepage();
            System. setProperty("webdriver.chrome.driver", "/Users/mrath/Downloads/chromedriver2");
            driver = new ChromeDriver();
            driver.get("https://www.google.com");
    
            // Check for Google logo
    
            if (driver.findElement(By.id(HPobj.google_logo())).isDisplayed())
            {
                System.out.println("Google logo is displayed");
            }
    
            // Type text in google search text box.
    
            driver.findElement(By.name(HPobj.google_searchTextBox())).sendKeys("Technology");
    
            // Click on google search button
    
            driver.findElement(By.name(HPobj.google_search_button())).click();
    
            driver.quit();
    
        }
    
    
    
    }




Page Object Model : Using "By" return type

3 Web loactors are represents with 3 methods having By return type.
google_logo
google_searchTextBox
google_search_button 


        package pageobjectmodel;
        import org.openqa.selenium.By;
        
        
        public class homepage {
        
        
            public By google_logo()

                return By.id("hplogo");

            public By google_searchTextBox()
            {
                return By.name("q");
            }
        
            public By google_search_button()
            {
                return By.name("btnK");
            }
        
        
        }
